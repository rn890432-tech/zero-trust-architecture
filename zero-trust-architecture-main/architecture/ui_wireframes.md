# UI Wireframes & Component Breakdown

## SOC Dashboard
- Event Timeline (chart)
- Queue Depth (gauge)
- Worker Health (table)
- Delivery Metrics (chart)
- Provider Health (status cards)
- Alerts Panel (severity, timestamp, description)
- Search/Filter Bar
- Role-based Access Controls
- Analyst View: drill-down, retry controls
- Executive View: summary metrics, SLA compliance
- Provider Status Panel: health, failover history
